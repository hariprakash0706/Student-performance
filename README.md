# Student Performance Prediction Using Machine Learning

## 📌 Overview
This project predicts whether a student will pass or fail based on attendance, internal marks, assignment scores, and study hours using Machine Learning algorithms.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## 📂 Dataset
- Student academic performance data
- Approximately 500 records

## 🔄 Workflow
- Data Collection
- Data Cleaning
- Feature Selection
- Model Selection
- Model Training
- Prediction

## 🤖 Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest

## 📊 Features
- Student performance prediction
- Data visualization
- Model comparison
- Accuracy evaluation

## 🚀 Output
Predicts whether a student is likely to **Pass** or **Fail** based on academic factors.
